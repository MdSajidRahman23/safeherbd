# SafeHer Bangladesh - AI-Driven Women Safety & Empowerment Platform



## Description
**SafeHer Bangladesh** is an AI-driven web platform designed to ensure the online and offline safety of women in Bangladesh. This project provides instant help in emergencies (SOS), safe route suggestions, and a supportive, moderated community forum.

This platform is being developed as a project for the Department of CSE, Daffodil International University.

---



## Features
Our platform's main features include:

* **SOS Alert System:** Send instant emergency alerts with the user's live GPS location.
* **Safe Route Suggestion:** Suggests safer routes to users based on crowd-sourced data and crime points.
* **AI-based Harassment Detection:** Automatically detects any harassing or offensive content in the forum and chat.
* **Mental Health Chatbot:** An AI chatbot providing 24/7 support for mental health and safety tips.
* **Women-Only Community Forum:** A secure, moderated community forum exclusively for female users.
* **Admin Dashboard:** A panel for admins to manage users, monitor SOS alerts, and view safety analytics.
